Message Bridge (메시지 중계소) | 시행일: 2025년 3월 5일

1. 수집하는 정보

본 앱은 메시지 전달 기능을 제공하기 위해 다음 정보를 처리합니다.

데이터 유형항목처리 방식
메시지SMS/MMS 발신자, 본문사용자가 설정한 채널로 전달 후 기기 내 로그 저장
알림앱 이름, 제목, 내용사용자가 설정한 채널로 전달 후 기기 내 로그 저장
이메일 주소Gmail 계정이메일 발송 인증용, 기기 내 저장
광고 ID기기 광고 식별자AdMob SDK가 광고 표시 목적으로 수집

2. 정보의 이용 목적

  • 수신된 SMS/알림 메시지를 사용자가 설정한 채널(Telegram, Slack, Email, SMS)로 전달
  • 전달 이력 기록 및 표시 (기기 내 저장)
  • Gmail API를 통한 이메일 발송
  • AdMob을 통한 배너 광고 표시

3. 정보의 저장 및 보안

  • 모든 데이터는 사용자 기기 내에만 저장됩니다. 별도 서버로 전송하지 않습니다.
  • 채널 자격증명(Telegram 봇 토큰, Slack 웹훅 URL)은 AES-256-GCM 암호화로 보호됩니다.
  • 외부 전송 시 모든 통신은 HTTPS로 암호화됩니다.
  • 전달 로그는 최대 500건, 30일까지 보관 후 자동 삭제됩니다.

4. 제3자 제공

본 앱은 개발자 서버로 사용자 데이터를 전송하지 않습니다. 다만, 사용자가 직접 설정한 외부 서비스로 메시지가 전달됩니다:

  • Telegram — Telegram Bot API를 통해 메시지 전달
  • Slack — Slack Webhook을 통해 메시지 전달
  • Gmail — Gmail API를 통해 이메일 발송
  • Google AdMob — 광고 표시를 위한 광고 ID 수집 (Google 개인정보처리방침)

5. 사용자 권리

  • 앱 설정에서 전달 기능을 켜거나 끌 수 있습니다.
  • 전달 규칙, 로그, 채널 설정을 앱 내에서 직접 삭제할 수 있습니다.
  • Gmail/Drive 계정 연결을 앱 내에서 해제할 수 있습니다.
  • 앱 삭제 시 기기 내 모든 데이터가 자동으로 삭제됩니다.
  • Android 설정에서 광고 ID를 재설정하거나 맞춤 광고를 비활성화할 수 있습니다.

6. 아동 개인정보

본 앱은 만 18세 이상 사용자를 대상으로 하며, 아동의 개인정보를 의도적으로 수집하지 않습니다.

7. 변경 사항

본 방침이 변경될 경우 이 페이지를 통해 공지합니다.

Privacy Policy

Message Bridge | Effective: March 5, 2025

1. Information We Collect

This app processes the following information to provide message forwarding functionality.

Data TypeDetailsProcessing
MessagesSMS/MMS sender, bodyForwarded to user-configured channels, logs stored on device
NotificationsApp name, title, contentForwarded to user-configured channels, logs stored on device
Email addressGmail accountUsed for email sending authentication, stored on device
Advertising IDDevice ad identifierCollected by AdMob SDK for ad display

2. How We Use Information

  • Forward received SMS/notification messages to user-configured channels (Telegram, Slack, Email, SMS)
  • Record and display forwarding history (stored on device)
  • Send emails via Gmail API
  • Display banner ads via AdMob

3. Data Storage and Security

  • All data is stored only on the user's device. No data is sent to our servers.
  • Channel credentials (Telegram bot tokens, Slack webhook URLs) are protected with AES-256-GCM encryption.
  • All external communications use HTTPS encryption.
  • Forwarding logs are automatically deleted after 500 entries or 30 days.

4. Third-Party Services

This app does not transmit user data to the developer's servers. However, messages are forwarded to external services configured by the user:

  • Telegram — Messages forwarded via Telegram Bot API
  • Slack — Messages forwarded via Slack Webhook
  • Gmail — Emails sent via Gmail API
  • Google AdMob — Advertising ID collected for ad display (Google Privacy Policy)

5. Your Rights

  • You can enable or disable forwarding in the app settings.
  • You can delete forwarding rules, logs, and channel settings directly within the app.
  • You can disconnect Gmail/Drive accounts within the app.
  • Uninstalling the app automatically deletes all on-device data.
  • You can reset your advertising ID or opt out of personalized ads in Android settings.

6. Children's Privacy

This app is intended for users aged 18 and above. We do not knowingly collect personal information from children.

7. Changes

Any changes to this policy will be posted on this page.
